# fedora-avf-installer
fedora-avf-installer replaces the default Debian image shipped with Android's Terminal app with a complete Fedora install

# Usage
Download the script and run it to perform the install. The script will guide you through the installation process.
```sh
curl -L https://raw.githubusercontent.com/cillyvms/fedora-avf-installer/refs/heads/main/install.sh -o install.sh
sudo bash install.sh
```
